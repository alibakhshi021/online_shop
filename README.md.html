📦 معرفی پروژه

این پروژه یک برنامه وب مبتنی بر Django است که با استفاده از Docker قابل اجرا در محیط‌های توسعه، تست و تولید می‌باشد. این پروژه شامل یک ماژول به نام account_module است که وظیفه مدیریت حساب‌های کاربری را دارد.

📁 ساختار پروژه

.
├── Dockerfile
├── docker-compose.yml
├── docker-compose-prod.yml
├── docker-compose-test.yml
├── requirements.txt
└── account_module/
    ├── admin.py
    ├── apps.py
    ├── formes.py
    ├── models.py
    ├── tests.py
    ├── urls.py
    ├── views.py
    └── migrations/

🚀 شروع به کار

پیش‌نیازها

  • Python نسخه ۳.۱۰ یا بالاتر
  • Docker و Docker Compose
  • Git

نصب و اجرا

  1. ابتدا مخزن پروژه را کلون کن: 
    git clone https://github.com/alibakhshipy/online_shop.git
cd your-repo-name
  2. اجرای کانتینرها با Docker: 
    docker-compose up --build
  3. برنامه از طریق آدرس زیر در دسترس خواهد بود: 
    http://localhost:8000/

🧪 اجرای تست‌ها

برای اجرای تست‌ها از فایل docker-compose-test.yml استفاده کن:

docker-compose -f docker-compose-test.yml run web python manage.py test

  • Gunicorn (برای محیط production)
